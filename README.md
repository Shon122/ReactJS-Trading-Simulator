# ReactJS-Trading-Simulator
- Built using ReactJS + Node + Express + FinancialModelingPrep API.
- Based on real historical prices for multiple timeframes and over 100 stocks/coins.
- Option to toggle Crypto/Forex/Stocks categories on and off. 
- All-time best score and current score will be saved after the user has logged off.
